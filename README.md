# KCG Deck Maker

神椿TCGデッキ構築支援アプリ（非公式）

## 技術スタック

- **フロントエンド**: Vue.js (TypeScript)
- **ビルドツール**: Vite
- **パッケージマネージャー**: Bun

## 開発環境のセットアップ

1.  リポジトリのクローン
    ```bash
    git clone https://github.com/suika-lunch/kcg-deck-maker.git
    cd kcg-deck-maker
    ```
2.  依存関係のインストール
    ```bash
    bun install
    ```

## 開発コマンド

- **リンタの実行**:
  ```bash
  bun lint
  ```
- **フォーマッタの実行**:
  ```bash
  bun format
  ```
- **開発サーバの起動**:
  ```bash
  bun dev
  ```
